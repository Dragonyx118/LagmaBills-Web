GET THE APP

Take Control

LagmaIpp is the cross-platform .NET MAUI app that drives the robot: dual joysticks, arm sliders, drone control and live camera stream over Tailscale.

LagmaIpp — actively developed, open source
🪟

Windows

.NET MAUI desktop build. Connects to the robot over Tailscale or your local network.

Build from source · .exe coming soon ⬇ Get it on GitHub No signed installer yet — clone the repo and build with Visual Studio / dotnet build.
🤖

Android

Same app, same controls — built for tablets and phones with the .NET MAUI Android target.

Build from source · .apk coming soon ⬇ Get it on GitHub Check the repo's Releases tab for a packaged APK once one is published.
🍏

iOS

Technically possible with .NET MAUI — just needs an Apple developer account we don't have yet.

Not available Unavailable If you want to help port/sign a build, open an issue on GitHub.

What you'll need

📡 Same network as the robot, or a Tailscale connection
🔌 Raspberry Pi 4B running the LagmaBills services
🎮 A gamepad/Bluetooth controller (optional, but recommended)
🛠️ .NET 8 SDK if you're building from source
