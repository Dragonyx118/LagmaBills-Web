THE PEOPLE

Who Built LagmaBills

A student project born in an Italian high school computer science lab — with a lot of late nights and a healthy amount of solder smoke.

🛠️

Daniele Cerioli

Lead Developer

Designs and builds the robot's hardware: mecanum chassis, robotic arm, custom KiCad PCBs and the cross-platform control app.

Python C++/Arduino C# MAUI KiCad
🎤

Simone Panelli

Father of Nova

Built Nova, the robot's voice assistant: wakeword detection, speech-to-text, local LLM and text-to-speech, fully offline.

Voice AI Python ONNX / Whisper
🔧

Andrei Corduneanu

DoomScroller

Testing Assembly Brainrot-certified

Open to Contributions

LagmaBills is fully open source. Whether you want to add a new sensor driver, improve the occupancy grid algorithm, train a better wakeword model, or port the MAUI app to iOS — pull requests are welcome. Check the GitHub repository for open issues and the roadmap.
